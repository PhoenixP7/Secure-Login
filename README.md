# Group 3
A web application that redesigns the login page for a school website, 
ensuring only authenticated users can gain access.

## Product backlogs 
    1. Email and password sign-in requirement
    2. Teacher and admin tab
    3. Additional security - memorable word
    4. Password reset function
    5. Lock user account after 3 failed log-in attempts
    6. Upgrade security measures to MFA for admins and teachers
    7. Include a tracking functionality for new sign-ups for admin users to review
    8. Add a parent log-in tab which only requires a memorable word as a security measure


## Sprint backlogs
    1. Create the MVP for a secure log-in webpage for a school. This includes the admin and teacher log-in, security question requirement and user page which you are redirected to when signing in.
    2. Upgrade CSS frontend for the webpage, add password reset functions
